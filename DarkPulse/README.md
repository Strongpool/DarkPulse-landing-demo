# DarkPulse

Демонстрационный лендинг с переключением тёмной темы.

## Что умеет проект

- Переключение светлой и тёмной темы с сохранением в localStorage
- Модальное окно покупки тарифа (определяет, какой тариф был выбран)
- Полностью адаптивный дизайн (работает на телефоне и планшете)
- Чистая вёрстка на HTML + CSS + Vanilla JavaScript

## Технологии

- HTML5
- CSS3 (Flexbox, CSS Variables, Media Queries)
- Vanilla JavaScript (DOM, localStorage, события)

## Запуск

Просто открой файл `main.html` в браузере.  
Или запусти локальный сервер:
```bash
python3 -m http.server 8000
```
